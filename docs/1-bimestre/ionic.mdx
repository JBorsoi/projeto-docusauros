# Desenvolvimento de Aplicativos Móveis Ionic

Na matéria de Desenvolvimento de Aplicativos Móveis com Ionic, aprendemos a utilizar o Balsamiq para realizar a prototipagem do nosso aplicativo.

Para nosso aplicativo "Condo-Link" fizemos a prototipagem de 6 telas: Login; Home; Perfil; Reservar; Minhas Reservas; Notificações; Reclamações e Contas. Além do design, no Balsamiq conseguimos colocar ações, como por exemplo, quando clicar no ícone "Perfil", o app irá para tela "Perfil"; quando clicar no botão "voltar", o app redireciona o usuário para a tela "Home".
