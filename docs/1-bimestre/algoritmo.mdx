# Lógica de Programação e Algoritmos

Lógica de programação é a organização de uma sequência de instruções com a finalidade de resolver um problema, ou de criar um software / aplicação.

A lógica de programação é o conhecimento anterior a qualquer outro quando falamos em desenvolvimento web, pois, é a partir dele que os aprendizados futuros, como por exemplo o das linguagens de programação, serão elaboradas.
Cada linguagem tem suas próprias particularidades, como sua sintaxe, seus tipos de dados e sua orientação, mas a lógica por trás de todas é sempre a mesma.

Em outras palavras, dominar a lógica de programação significa  tornar-se um programador completo, seja em front-end ou em back-end.
